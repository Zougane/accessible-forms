Label in name test

This page has a series of tests related to Success Criterion 2.5.3 Label in Name.

Keeping in mind that accessible names are generated in the following priority order:

  1. If present, use aria-labelledby value.
  2. Otherwise, if present, use aria-label value.
  3. Otherise, if present, use link content.
  4. Otherwise, if present, use title value.
  5. If none of the above are provided, there is no accessible name.

In the examples below, the label (in this case the visible test associated with the link) may not match the accessible name - the name assigned in the accessibility tree.
